全球海缆拓扑可视化系统 (Global Submarine Cable Topology System)

> Vue.js + ECharts 打造的海缆运行态势与故障打点一体化可视化开源项目，覆盖全场景视图切换、智能故障定位、专业测距打点等核心能力，适配多终端运维与展示场景。

📌 Project Overview（项目概述）

本项目是基于 Vue.js + ECharts + Python 构建的开源海缆拓扑可视化与运维辅助系统，核心用于海缆信息全生命周期管理、拓扑可视化呈现、故障精准定位与运维流程辅助，可无缝嵌入海缆故障处理流程。

针对海缆运维领域「故障定位模糊、运维流程繁琐、场景覆盖不足」三大核心痛点，提供沉浸式可视化、高效交互操作、全流程闭环管理的一体化解决方案，兼顾日常运维实用性与接待展示专业性，可直接复用至各类海缆运维场景，支持二次开发与定制化扩展。

✨ Key Features（核心特性）

1. 9大模式全覆盖：支持紧凑/大屏模式、2D普通/2D瓦片/3D模式、聚焦/打点模式、紧凑/大屏实况模式，一键切换适配不同运维与展示场景，视觉与交互体验全程一致。

2. 沉浸式可视化体验：2D/3D地球自由切换，内置多套高清纹理（地形/白昼/夜光/8K），3D模式支持自转、球面插值、光效调节，2D瓦片模式支持高分辨率缩放与经度循环，还原海缆真实分布态势。

3. 智能故障管控：故障列表按严重度排序，支持一键聚焦故障海缆、叠加全量故障形成热力层，搭配自定义故障面板，实现故障发现-定位-记录-回溯的全闭环管理。

4. 专业打点测距：仅允许沿海缆走线打点，支持30/80km偏移容差兜底，实时计算沿线距离，绿线+预览点常驻显示，离线时自动回退至最近有效打点，确保故障定位精准无误。

5. 高效交互设计：支持分栏拖拽调整、列表拖动重排，布局参数本地持久化，海缆/登陆站双向聚焦，Tooltip芯片一键跳转，操作流畅且降低运维复杂度。

6. 多场景适配：默认紧凑模式适配小屏，大屏模式适配监控场景，实况模式支持自动自转与全屏展示，兼容多浏览器，Edge浏览器专项优化确保稳定运行。

7. 全局i18n支持：完善的中英文双语适配，覆盖所有按钮、标签、提示文案，支持地图版本（亚太中心/标准版）切换，坐标自动转换确保显示一致。

8. 灵活部署适配：支持免登录演示模式、故障数据接口兜底，可对接自有故障接口，提供离线包适配计划，满足公网演示、本地评估、生产部署等多场景需求。

🚀 Quick Start（快速开始）

环境要求

- 运行环境：Nginx/Apache 静态服务器，或 Python 3.x 环境

- 依赖框架：Vue.js 2.x、ECharts 5.x + ECharts-GL 2.x、Element UI

- 浏览器兼容：Chrome、Firefox、Edge（EdgeHTML版本专项优化）

安装部署

# 1. 克隆项目（假设开源仓库地址为https://github.com/kongweihao/wsc.git）
git clone https://github.com/kongweihao/wsc.git

# 2. 进入项目目录
cd wsc

# 3. 启动静态服务（二选一）
# 方式1：Python静态服务
python -m http.server 8080

# 方式2：Nginx/Apache部署
将项目文件放入Web根目录（如/www/wwwroot/127.0.0.1/wsc/）

访问项目

浏览器打开以下地址即可访问：

- 本地预览：http://localhost:8080/wsc/index.html

- 服务器访问：http://<server-ip>/wsc/index.html

- 免登录演示：http://<server-ip>/wsc/index.html?developmode=1（跳过登录校验，启用演示故障数据）

📖 Usage（使用指引）

核心模式切换

- 模式入口：右下角 Speed Dial 控制台（主按钮含2D/3D切换、全屏等核心操作，折叠面板提供详细设置）。

- 视图切换：点击控制台「2D/3D切换」，3D模式可选择纹理、开启自动自转，2D模式可切换底图与瓦片模式。

- 布局切换：控制台一键切换紧凑/大屏模式，拖拽分栏分隔条可调整宽度/高度，布局自动本地保存。

核心操作流程

1. 筛选：左侧面板（紧凑模式为抽屉）勾选权益/类型/状态，通过洲别、里程下拉筛选，支持拼音/括号匹配搜索。

2. 聚焦：点击海缆、登陆站或故障列表项，自动聚焦目标对象，再次点击可退出聚焦，跨经线聚焦自动校正避免偏移。

3. 打点：故障编辑面板点击「打点」，悬停/点击海缆走线即可生成预览点与测距数据，点击确认后锁定预览，支持继续打点或保存。

4. 实况模式：控制台点击「开启PC/大屏实况模式」，自动执行全屏、切换3D、叠加故障、开启自转流程。

🔧 Tech Stack（技术栈）

- 前端：Vue.js 2.x、原生 JavaScript (ES6+)、Element UI

- 可视化：ECharts 5.x、ECharts-GL 2.x（3D可视化）

- 后台：Python 3.x、Thinkphp5.1

- 部署：Docker、Nginx/Apache 静态服务器

- 其他：localStorage（布局持久化）、Haversine 算法（测距）、球面插值（3D曲线优化）

📂 Directory Overview（目录速览）

- index.html：页面骨架与紧凑/大屏模式核心结构

- css/app.css：全局样式、布局与组件样式定义

- js/app.js：业务逻辑、地图渲染、打点测距与故障联动核心代码

- seacable_data/：海缆、登陆站数据与纹理资源存储目录
